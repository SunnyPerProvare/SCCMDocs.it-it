## <a name="improvements-to-console-connections-node"></a>Miglioramenti al nodo Connessioni di console
<!--4923997, 4951240 -->
Nel nodo **Connessioni di console** la colonna **Last Console Heartbeat** (Ultimo heartbeat colonna) ha sostituito la colonna **Ora dell'ultima connessione**. La colonna **Last Console Heartbeat** (Ultimo heartbeat colonna) offre agli amministratori maggiori informazioni per determinare quali connessioni della console sono attualmente attive. Quando Ã¨ aperta una console di Configuration Manager, viene eseguito un controllo ogni 10 minuti. Se durante il controllo la console viene eseguita in primo piano, la colonna **Last Console Heartbeat** (Ultimo heartbeat colonna) viene aggiornata.
